Welcome to my new blog powered by **Jekyll**! 

这是一个用 Jekyll 构建的静态博客。Jekyll 是一个简单、静态网站生成器，非常适合个人博客和项目文档。

## 为什么选择 Jekyll？

1. **简单** - 只需 Markdown 即可写文章
2. **快速** - 静态生成，访问速度快
3. **免费托管** - 可以直接部署到 GitHub Pages
4. **自定义** - 完全掌控代码

## 快速开始

```bash
# 本地开发
docker-compose up

# 写新文章
# 在 _posts 目录创建 YYYY-MM-DD-title.md 文件
```
